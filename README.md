# LMS API Doc
* https://unialto.github.io/lms-api-doc/

### Install node
```shell
brew install node@20
```

### Install apiDoc
```shell
npm i apidoc@0.29.0 -g
```

### Generate HTML
```shell
apidoc -i src -o docs -t template
```
